## Environment Variables
### Server
* `INSTANCE_LOCATOR`
* `SECRET_KEY`

### Client
* `INSTANCE_LOCATOR`

## Server
Run the auth server via:
```
cd api/
npm i
npm start
```

## Client
Create a user/room via the inspector in the Pusher Dashboard.

Run the client via:
```
cd api/
npm i
node index.js userId roomId
```
